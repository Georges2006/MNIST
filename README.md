MNIST pour la classification de l'ecriture manuscrite par des Réseaux de neurones artificiels (ANN)
La base de données utilisée est celle de MNIST (http://yann.lecun.com/exdb/mnist/ ). 
Ce problème de reconnaissance de l'écriture manuscrite est un problème difficile, et un bon test pour les algorithmes d'apprentissage. 
La base MNIST est un test qui regroupe 60000 images d'apprentissage et 10000 images de test. 
Ce sont des images en noir et blanc, normalisées centrées de 28*28 pixels.
Nous allons utiliser les réseaux multicouches pour résoudre ce problème de reconnaissance
